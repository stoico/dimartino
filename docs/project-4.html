Startup Map - Knockout.js
Demo
A reactive web application
A single-page application in Knockout.js.
Highlights the location of startups in the area and displays relative information about each. Integrates various data points from the Google Maps API and the Lever API.
An MVVM pattern is implemented (through ES6 and Knockout.js ) to create a reactive application able to handle the various functionality: list, filters, and information on the page subject to changing state.
I developed the same app in Vue.js as a means of learning and comparing different frameworks.
